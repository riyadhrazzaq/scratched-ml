5. Multinomial Naive Bayes
6. Gaussian Naive Bayes
7. Support Vector Machine
8. Decision Tree Classifier
9. Decision Tree Regressor
10. AdaBoost
11. Non-negative Matrix Factorization
12. Hierarchical Clustering (agglomerative)
